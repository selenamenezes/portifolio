Sobre

Olá! Sou Selena Menezes, estudante de Análise e Desenvolvimento de Sistemas, atualmente no último período, e atuo como estagiária de Análise de Dados na Neoenergia Pernambuco. Tenho interesse em tecnologia, análise de dados e no desenvolvimento de soluções que auxiliem na tomada de decisões e na otimização de processos. Ao longo da minha trajetória, venho desenvolvendo conhecimentos em Python, SQL, PostgreSQL, Docker, Git/GitHub, Power BI, Power Automate e Excel, buscando continuamente aprender novas ferramentas e aplicar boas práticas da área.

Meu objetivo é construir uma carreira sólida em dados, ampliando meus conhecimentos em análise, automação e engenharia de dados. Busco oportunidaes que me permitam aplicar na pratica o que aprendo, enfrentar novos desafios e contribuir para o crescimento das organizações por meio de soluções baseadas em dados.
Estou em constante aprendizado e acredito que a combinação entre curiosidade, dedicação e experiência prática é fundamental para evoluir como profissional de tecnologia.

SofIA

Python • Inteligência Artificial • RAG

Aplicação que utiliza IA com abordagem de (RAG) para responder perguntas com base em documentos fornecidos pelo usuário, garantindo respostas contextualizadas e confiáveis.

✓ Entrada: recebe e processa arquivos diversos como base de conhecimento.
✓ Busca Semântica: identifica trechos relevantes com base no significado das perguntas.
✓ Respostas: utiliza IA para responder com contexto recuperado (RAG).
✓ Confiabilidade: reduz erros ao fundamentar respostas em dados reais do usuário.
GitHub

  2. Configure variáveis em .env
  3. Execute: python main.py

NYC Taxi Data Pipeline

Pipeline de Dados • Qualidade de Dados • Processamento em Lote

Pipeline em Python que transforma e organiza grandes volumes de dados de transporte, aplicando limpeza, cálculos e enriquecimento, com saída eficiente em Parquet para análise.

✓Processamento : leitura de grandes volumes com iter_batches para otimização de memória
✓ Limpeza e transformação: tratamento de nulos, remoção de duplicados e padronização de dados
✓ Enriquecimento: mapeamento de zonas e cálculo de métricas como distância e pagamento
✓Saída: escrita em Parquet com compressão e schema consistente
GitHub

Automação de Dados SAP

Automação • Orquestração • Pipeline de Dados

Sistema de automação em Python para extração de dados do SAP via automação de interface, organização de arquivos e distribuição em diretórios estruturados. Executa rotinas programadas ao longo do dia, incluindo backup, versionamento e atualização de bases para consumo analítico.

✓Automação : extração de dados via scripts automatizados com PyAutoGUI e execução programada
✓Gestão de arquivos: organização, cópia e versionamento de bases em múltiplos diretórios
✓Orquestração: agendamento de processos recorrentes com controle de execução ao longo do dia
✓Backup e histórico: criação automática de versões diárias e armazenamento estruturado de dados
