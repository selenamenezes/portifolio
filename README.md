# Portfólio Selena Menezes (site estático)

## Como usar (editar rápido)
1. Abra `index.html`.
2. Substitua:
   - `https://github.com/SEU_USUARIO` e `https://www.linkedin.com/in/SEU_USUARIO`
   - `mailto:SEU_EMAIL@exemplo.com`
   - Nos cards de projetos: títulos, meta, bullets e links do GitHub.
3. Para adicionar mais projetos: copie um `article.projectCard` e cole dentro do `div#projectGrid`.

## Estrutura
- `index.html` — layout e conteúdo
- `styles.css` — estilos (tema azul)
- `script.js` — pequenas interações (mostrar/ocultar detalhes)

## Como hospedar (rápido)
Como é um site estático, você pode hospedar em:
- GitHub Pages
- Netlify
- Vercel (static)
- qualquer serviço que aceite HTML/CSS/JS

(Se quiser, posso te passar um passo-a-passo para o serviço que você preferir.)
